Para executar o aplicativo simulador de Pipeline, basta acessar src/html/ e abrir
o arquivo App.html no navegador Google Chrome. Clique no botão Upload, e escolha
o arquivo assembly.txt. Após realizado o upload do arquivo, basta clicar no botão
Start, para então ser iniciada a simulação. A cada clique no botão Next Stage, é iniciada
um novo estágio do Pipeline.
